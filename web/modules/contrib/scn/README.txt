Simple Comment Notify
https://www.drupal.org/project/scn
==============================================

INTRODUCTION
------------

Simple Comment Notify is a lightweight tool to send notification e-mails to 
admins/moderators when user create comment.

REQUIREMENTS
------------
comment (core)

INSTALLATION
------------

Install as you would normally install a contributed Drupal module. Visit:
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
for further information.

CONFIGURATION
-------------

Go to admin/config/system/scn and apply settings
